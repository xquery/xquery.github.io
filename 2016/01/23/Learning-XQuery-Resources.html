Learning XQuery Resources

a list of resources to learn XQuery

  • curated list of Learning XQuery resources

  • download an XQuery processor (Saxon, XQilla, Zorba)

  • download a database that supports XQuery and start learning (MarkLogic, exist DB, basex)

  • follow @XQuery

  • goto great conferences (XML Prague, XML London, XML Amsterdam, XML Summer School, Balisage)

  • free training ! at MarkLogic University

  • http://x-query.com/pipermail/talk/ [xquery mailing list]

  • XQuery at StackOverflow

  • Priscilla Walmsley’s excellant uptodate book 'XQuery: Search Across a Variety of XML Data'

give me a shout (@_james_fuller) if you want me to add anything.
